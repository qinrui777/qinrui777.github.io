tr command

May 07 2019

tr 是 translate(转换)的简写，可以对来自标准输入的内容进行字符 替换、删除、以及重复字符压缩。

tr [options] set1 set2
来自sdtin的输入字符会按照位置从 set1 映射到 set2 (set1中的第一个字符映射到set2中的第一个字符，以此类推)，然后将输出写入stdout
