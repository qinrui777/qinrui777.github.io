5.1 注释

井号（#）在Makefile中表示注释。

5.1 回声（echoing）

正常情况下，make会打印每条命令，然后再执行，这就叫做回声（echoing）。

在命令的前面加上@，就可以关闭回声。

命令前增加了一个 - 符号，这个符号的目的就是如果这条命令执行失败了继续执行，不影响后续命令的执行。
`bash

➜ cat Makefile

all: stage1 stage2

stage1: ;@echo “run stage1…..”;
@echo “I am stage1”
-cat xxx
stage2:
echo “I am stage2”

#执行
➜ make
run stage1…..
I am stage1
cat xxx
cat: xxx: No such file or directory
make: [stage1] Error 1 (ignored)
echo “I am stage2”
I am stage2
